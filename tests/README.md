
# Testing

The testing is done using the python unit test framework `pytest`.


## Grids

There are currently 4 testing subjects:

  - Complex grids - ComplexGrid
    - Properties
    - Copy
    - FFT
    - FFT speed

  - Real grids - RealGrid
    - Properties
    - Copy
    - FFT

  - 1D grids - ComplexGrid1D, RealGrid1D
    - Properties
    - Copy
    - FFT

  - 2D grids - ComplexGrid2D, RealGrid2D
    - Properties
    - Copy
    - FFT
